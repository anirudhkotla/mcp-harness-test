# Todo App

A simple todo application built with HTML, CSS, and JavaScript.

## Features

- Add new todo items
- Delete todo items

## How to Use

1. Enter a todo item in the input field
2. Click the "Add" button or press Enter
3. To delete a todo item, click the "Delete" button next to it

## Installation

No installation required. Simply open the `index.html` file in your web browser.
